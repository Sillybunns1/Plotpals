# PlotPals 

This version removes AI and adds Supabase account/cloud sync wiring.

## Included

- Multiple series
- Book-first project structure
- Manuscript editor
- Chapter planner
- Characters
- Plot threads
- Timeline
- Worldbuilding
- Relationships
- Export TXT
- Export Word-compatible DOC
- Export PDF through browser print
- Export Series Bible DOC
- JSON backup/restore
- Rich text formatting
- Full-screen writing mode
- Chapter navigation
- Auto-save local + cloud
- Writing statistics
- Full manuscript export
- DOCX export
- PDF export
- Series bible export


1. Click Account.
2. Sign up or sign in.
3. Create your series/book.
4. Write in the Manuscript Editor.
5. Use Sync Now if you want to force cloud save.
6. Use Export for manuscript DOCX/PDF and series bible DOCX/PDF.

## Important Note About DOCX

The DOCX export is Word-compatible HTML saved with a `.docx` extension. Microsoft Word and Google Docs should open it. For a production-grade app, a server-side DOCX generator would be the next upgrade.
